# Pesquisa sobre Experiências com Crianças com Autismo

Este projeto consiste em uma pesquisa digital voltada para coletar informações sobre experiências educacionais e terapêuticas com crianças com autismo.

## Estrutura do Projeto

A pesquisa está organizada em três formulários específicos para diferentes públicos:

- **Pais/Responsáveis**: Formulário direcionado a familiares de crianças com autismo
- **Professores**: Formulário para educadores que trabalham com alunos autistas
- **Outros Profissionais**: Formulário para terapeutas, psicólogos e outros profissionais

## Funcionalidades

- Formulários responsivos e acessíveis
- Armazenamento local de respostas (localStorage)
- Área administrativa para visualização e exportação de dados
- Exportação em formato CSV para análise posterior

## Como Acessar

A pesquisa está disponível online através do GitHub Pages:
[Link da Pesquisa](https://seu-usuario.github.io/autism-survey/)

## Área Administrativa

Para acessar os dados coletados:
1. Clique no link "Área Administrativa" no rodapé da página inicial
2. Use a senha: DADOS PROTEGIDOS PELA LEI DE PROTECAO DE DADOS, SOMENTE OS ADMINISTRADORES DO PROJETO TERAO ACESSO.

## Tecnologias Utilizadas

- CSS3
